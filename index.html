How I Created This Clock

I created a "div" with id set it to "clock". then defined a function "myClock" that gets the current time using the "Date" object, formats the time into hours, minutes, and seconds using string manipulation, and displays the formatted time in the div element. The "setInterval" method is used to call the "myClock" function every second to update the clock. The "myClock" function is also called once to initialize the clock. then i added some css with external file so it looks good.
